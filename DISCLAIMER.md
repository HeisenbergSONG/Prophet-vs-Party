1. 本平台比较共产党宣传、基督教传教、伊斯兰教达瓦（传教）三类话语的修辞技巧与心理机制，**不构成任何政治、宗教或道德立场**。

## English

**Prophet vs Party** is an educational tool for rhetorical analysis only.

1. The platform compares discourse techniques across three traditions; it **does not endorse or condemn** any political or religious view.
2. All cases are **publicly verifiable quotations** with cited sources.
3. Automated analysis outputs are **heuristic estimates**, not authoritative judgments.
4. Users are responsible for their own use of this tool. Report errors via GitHub Issues.
