# 程序设计方案：宣传 vs 传教对比分析平台

**名称**：  
**Prophet vs Party**（先知 vs 政党）  
或中文名：**“话语之战”——共产党宣传与宗教传教对比分析平台**

## 定位
一个**中立、教育性**的交互式工具，帮助用户系统地对比**共产党（以中共为代表）宣传话术**与**基督教、伊斯兰教传教话术**的异同。  
核心目标是**分析修辞技巧、心理机制、动员策略**，而非评判优劣。

## 核心功能模块

### 模块A：案例库（数据库）
- 分类收录真实历史/当代案例：
  - **共产党宣传**：中共口号（“为人民服务”“中国梦”“共同富裕”）、文革标语、官方媒体标题、领袖讲话片段。
  - **基督教传教**：福音布道词、耶稣语录、现代布道（如比利·格雷厄姆）、教会宣传片。
  - **伊斯兰教传教**：古兰经经文、哈迪斯、现代达瓦（Da’wah）演讲、社交媒体传教内容。
- 每条案例标注：**来源、时间、受众、核心诉求**。

### 模块B：多维度对比引擎
用户可选择任意两条（或三条）案例进行对比，系统自动生成分析报告：

| 对比维度           | 共产党宣传典型特征               | 基督教传教典型特征               | 伊斯兰教传教典型特征             |
|--------------------|----------------------------------|----------------------------------|----------------------------------|
| **核心叙事**      | 历史必然 + 阶级/民族复兴        | 救赎 + 个人罪与恩典             | 顺服真主 + 末日审判             |
| **情感诉求**      | 希望 + 恐惧（阶级敌人）         | 爱 + 恐惧（地狱）               | 敬畏 + 希望（乐园）             |
| **权威来源**      | 党/经典著作/领袖                 | 《圣经》/耶稣/教会传统           | 《古兰经》/先知穆罕默德         |
| **动员方式**      | 集体行动 + 组织纪律              | 个人悔改 + 教会共同体           | 社区（乌玛） + 礼仪实践         |
| **敌人形象**      | 阶级敌人/帝国主义/修正主义      | 魔鬼/罪/异教徒                  | 撒旦/异教徒/背教者              |
| **未来许诺**      | 人间天堂（共产主义）             | 天国/永生                       | 乐园/末日审判                   |
| **传播媒介**      | 官方媒体、宣传画、社交媒体      | 讲道、圣经、电影、社交媒体      | 清真寺、社交媒体、APP           |

### 模块C：可视化与分析工具
- **并排对比视图**：原文 + 翻译 + 高亮关键词
- **词云/关键词提取**：自动提取高频词（“斗争”“救赎”“顺服”等）
- **雷达图**：展示每个案例在“情感强度”“权威性”“希望指数”“恐惧指数”上的得分
- **时间轴**：展示不同时期话术演变
- **相似度算法**：用自然语言处理（NLP）计算两条话术的相似度（基于修辞结构）

### 模块D：教育与互动
- **技巧拆解**：点击任意案例，弹出“话术技巧标签”（如：二元对立、末世论、救世主叙事、群体归属等）
- **用户上传**：允许用户上传自己的案例，系统自动分析并对比
- **测验模式**：猜这是共产党还是宗教的话术（带解释）

## 技术架构建议
**推荐方案**：**Web应用**（易于分享和扩展）

- **前端**：React / Vue.js + Tailwind
- **后端**：Python（FastAPI / Django）
- **数据库**：PostgreSQL + 向量数据库
- **AI/NLP**：使用大模型进行文本分析
- **部署**：Vercel / Railway / 阿里云

## 扩展方向
- 多语言支持（中英阿）
- 视频/音频片段对比
- 历史版本演变追踪
- 生成“混合话术”实验

---

**说明**：本设计为教育/学术用途，保持中立分析。
